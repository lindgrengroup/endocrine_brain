# Phenotype creation

Scripts to define the neuroendocrine phenotypes:

1. pheno_covar_create_extended.R - script to define and extract phenotypes and covariates for use in GWAS
2. pheno_covar_create_WMGM_510_03.R - script to define and extract additional phenotypes and covariates describing white and grey matter for use in GWAS
3. epi_plots_brain_age_sex_headsize.R - script to generate plots to describe the phenotypes including the distribution across age and sex.
