# Hormone Associations

Scripts in this folder to look at genetic links to hormones:

1. hypergeometric_testosterone.R - test for enrichment of testosterone variants using hypergeometric test
2. gc_corrplot.R - make correlation plots for genetic correlation with hormones and infertility phenotypes
3. MR_analysis.R - perform Mendelian randomisation using sumstats from [hormone and infertility GWAS](https://www.medrxiv.org/content/10.1101/2024.03.19.24304530v1)
