# Cell-type enrichment analysis

Scripts in this folder to peform and process cell-type enrichment analysis. 

1. cellect_hypothalamic_plots.R - Process and plot results of CELLECT/CELLEX cell-type enrichment analysis across hypothalamic cell types. Data was downloaded from [here](https://storage.googleapis.com/linnarsson-lab-human/adult_human_20221007.loom) from Siletti et al., Transcriptomic diversity of cell types across the adult human brain. Science 382(6667), 7046 (2023).
2. sumstats_format_z_score_pval_ovamap.R - format he REGENIE output ready for use in CELLECT/CELLEX with the ovamap dataset
3. cellect_ova_plots.R - Process and plot results of CELLECT/CELLEX cell-type enrichment analysis across ovarian cell types within an in-house Ovarian cell atlas, OvaMap.
4. depict_table.R - Process output of tissue, pathway and cell-type enrichment analysis performed using DEPICT.
