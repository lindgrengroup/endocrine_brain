# Phenome-wide association analysis studies (PheWAS)

Scripts in this folder to perform PheWAS using UK Biobank and [Big40](https://open.win.ox.ac.uk/ukbiobank/big40/) traits:

1.  big40_phewas.R - Run phenome-wide association using Big40 database (https://open.win.ox.ac.uk/ukbiobank/big40/) and generate associated plots.
2.  run_phewas_ukbb.R - Run phenome-wide association using UKBB ICD10 codes and generate associated plots.
