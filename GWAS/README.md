# Genome-wide Association Studies (GWAS) and associated analyses

Scripts in this folder to perform GWAS and downstream associated analyses:

1.  regenie_step1.sh - run step 1 of GWAS in REGENIE.
2.  regenie_step2.sh - run step 2 of GWAS in REGENIE.
3.  gwas_plot.R - parsable script to plot manhattan plot and qqplot from filtered GWAS output.
4.  gwas_miami.R - script to plot miami plots to display sex-specific results for each of the four neuroendocrine phenotypes.
5.  gwas_filter.R - parsable script to filter REGENIE GWAS results for info score, Hardy-Weinberg equilibrium, minor allele frequency and multiallelic alleles.
